# Smart Expense Tracker with Analytics Dashboard

## 📌 Description
This is a full-stack web application to track daily expenses, manage budgets, and visualize spending using charts.

## 🚀 Features
- User authentication (Login/Register)
- Add, edit, delete expenses
- Category-wise tracking
- Monthly analytics dashboard
- Budget alerts

## 🛠 Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB
- Charts: Chart.js

## 📂 Project Structure
- frontend/ → User interface
- backend/ → Server and APIs
